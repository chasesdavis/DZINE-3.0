# Changelog

## 0.1.0

Initial D-ZINE 3.0 framework preview.

- Added `.dzine.html` parser and metadata extraction.
- Added prompt, goal, token, CSS, Tailwind, and compatibility Markdown exports.
- Added document and visual audit loops.
- Added multi-page site generation with `dzine site build`.
- Added route-by-route site audit with `dzine site audit`.
- Added source-backed Taste Graph and taste audit commands.
- Added generated product website with Home, Docs, Taste, Framework, and Showcase routes.
