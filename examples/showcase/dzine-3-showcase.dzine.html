Quality gates

Render, screenshot, audit, then ship.

The Goal 3 workflow captures mobile, tablet, and desktop screenshots, checks responsive overflow, contrast samples, target sizes, reduced-motion declarations, section completeness, and visible text sizing.

  • dzine screenshot captures viewport evidence.
  • dzine audit --visual writes JSON and Markdown reports.
  • dzine export --target all proves the source compiles into agent and framework artifacts.
